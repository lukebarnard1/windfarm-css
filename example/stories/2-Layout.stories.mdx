# Layout

A thing to put things in places.

By default, Layout acts as a generic `div` tag.

<Preview>
  <Story name='simple'>
    <Layout>
      <Text s_lg>Header</Text>
      <Text>
        { generateText({sentences: 2}) }
      </Text>
    </Layout>
  </Story>
</Preview>

## row

The `row` prop gives Layout `display: flex`, which by default arranges
children as equal columns. It will also remove any `margin` of children.

<Preview>
  <Story name='row'>
    <Layout row iflx_exp>
      <Text>col 1</Text>
      <Button c_p5>col 2</Button>
    </Layout>
  </Story>
</Preview>

<Preview>
  <Story name='row with an empty column'>
    <Layout row iflx_exp>
      <Button c_s5>col 1</Button>
      <Empty/>
      <Button c_s5>col 3</Button>
    </Layout>
  </Story>
</Preview>

<Preview>
  <Story name='row with five columns'>
    <Layout row iflx_exp>
      <Text>col 1</Text>
      <Text>col 2</Text>
      <Text>col 3</Text>
      <Text>col 4</Text>
      <Text>col 5</Text>
    </Layout>
  </Story>
</Preview>

## column

To arrange children in a column instead, use `col`.

<Preview>
  <Story name='col'>
    <Layout col>
      <Text>row 1</Text>
      <Text>row 2</Text>
      <Text>row 3</Text>
      <Text>row 4</Text>
      <Text>row 5</Text>
    </Layout>
  </Story>
</Preview>

## Empty

An empty element can be used to space rows or columns out.

<Preview>
  <Story name='empty'>
    <Layout col>
      <Text>row 1</Text>
      <Empty/>
      <Text>row 3</Text>
      <Text>row 4</Text>
      <Text>row 5</Text>
    </Layout>
  </Story>
</Preview>

## arrangement

Layout can also be used to position children within them.

<Preview>
  <Story name='arrangement'>
    <Layout col>
      <Layout row bgn>
        <Text>bgn</Text>
      </Layout>
      <Layout row mid>
        <Text>mid</Text>
      </Layout>
      <Layout row end>
        <Text>end</Text>
      </Layout>
    </Layout>
  </Story>
</Preview>


## padding

To apply padding to Layout, use the prefix `p_` followed by a two-letter size tag.
The larger the size, the greater the padding.

<Preview>
  <Story name='padding:xs'>
    <Layout p_xs>
      <Text s_md>p_xs</Text>
      <Text s_xs>
        { generateText({sentences: 10}) }
      </Text>
    </Layout>
  </Story>
</Preview>
